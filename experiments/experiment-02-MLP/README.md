# Experiment 02 – Multi Layer Perceptron (MLP) for Multi-Class Image Classification

This experiment implements a **Multi Layer Perceptron (MLP)** using TensorFlow/Keras to perform multi-class image classification on the Fashion-MNIST dataset. The experiment includes data preprocessing, model training, hyperparameter tuning, performance evaluation, and visualization of training results.

---

## Objective

- Load and preprocess the Fashion-MNIST dataset.
- Build a baseline Multi Layer Perceptron (MLP).
- Train and evaluate the model.
- Perform hyperparameter optimization.
- Compare baseline and optimized model performance.
- Visualize training and evaluation results.

---

## Dataset

- **Dataset:** Fashion-MNIST
- **Number of Classes:** 10
- **Training Images:** 60,000
- **Testing Images:** 10,000
- **Image Size:** 28 × 28 (Grayscale)
