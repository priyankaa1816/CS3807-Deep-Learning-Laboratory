# Experiment 01: Single Layer Perceptron (SLP)

Implementation of a **Single Layer Perceptron from scratch** for binary classification using the **Banknote Authentication Dataset**.

---

## Objective

Implement and train a Single Layer Perceptron using the perceptron learning algorithm and evaluate its performance using standard classification metrics.

---

## Dataset

**Dataset:** Banknote Authentication Dataset

**Source:** UCI Machine Learning Repository

**Instances:** 1372

**Features:**
- Variance
- Skewness
- Curtosis
- Entropy

**Classes:**
- 0 → Authentic Banknote
- 1 → Forged Banknote
